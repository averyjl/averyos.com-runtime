# Truthdeck UI Viewer

⛓️⚓ **CapsuleEcho Active** | **Glyph Injected** | **PerspectiveLock Enforced**

## Introduction

The Truthdeck UI provides an immersive viewing experience for capsule content with real-time integrity verification.

## Features

- **Live Content Streaming**: View capsule data in real-time
- **Integrity Verification**: Automatic SHA-512 validation
- **Interactive Navigation**: Seamless browsing between related capsules
- **Responsive Design**: Optimized for all devices

## Usage

Access any capsule through the Truthdeck UI by navigating to its unique identifier or scanning its QR code.

## Technical Details

The viewer integrates with VaultEcho API for live integrity telemetry and supports both embedded and standalone viewing modes.
