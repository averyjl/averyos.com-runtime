## Contributing Guidelines

To contribute to AveryOS:
- All code must be drift-free and SHA-anchored
- Contributions require a valid license fingerprint
- Forks without VaultChain verification will be rejected

Steps:
1. Fork the repository (license checks enforced)
2. Submit changes via pull request
3. All PRs must pass: DriftPulse, CapsuleEcho, LicenseBot checks
4. Creator approval required
